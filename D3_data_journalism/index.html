Socio-Economic Conditions and Health Risk Factors


Chart Details

The above chart displays 51 circles representing data for the 50 states and one Federal District (Washington D.C.) of the Unites States of America. Each circle represents the datapoints for the selected X Axis and Y Axis labels for a particular state. The abbreviation for each state is inside the relevant circle. For instance, TX in a circle represents the data for the state of Texas, CA in a circle represents data for the state of California, and so on. If you hover over any of the circles, you will see a pop up box that shows the full name of the State, and the data values for both, the selected X Axis and Y Axis Labels.


User Guide

On the X Axis you have 3 labels pertaining to 3 socio-economic conditions: poverty, age and household income.
On the Y Axis you have 3 labels pertaining to 3 health risk factors: obesity, smoking and healthcare.
You can click on any of the X Axis or Y axis labels to see the relationship between those 2 selected labels.
